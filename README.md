## Proyecto en desarrollo

### Sistema de monitoreo de inversión dual

Desarrollo de un dashboard financiero orientado a la toma de decisiones para un inversionista con dos tipos de activos:

- Un portafolio internacional de dividendos enfocado en generación de ingresos pasivos y libertad financiera.
- Una PyME de energía solar financiada con deuda y enfocada en generación de flujo de efectivo y beneficios fiscales.

El sistema permite monitorear indicadores financieros y generar recomendaciones automáticas en tiempo real relacionadas con reinversión, mantenimiento, apalancamiento y expansión de activos.
